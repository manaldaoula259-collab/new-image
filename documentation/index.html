üìã Prerequisites

Before you begin, ensure you have:
  • Node.js 18+ installed
  • MongoDB database (Atlas recommended)
  • Clerk account (for authentication)
  • Replicate API token (for AI processing)
  • Stripe account (for payment processing)
  • AWS S3 bucket (for image storage)
  • AWS credentials (Access Key ID and Secret Access Key)

‚ö° Quick Start

  1. Clone: Get the repository from GitHub
  2. Install: Run npm install to install dependencies
  3. Environment Setup: Configure all required API keys in .env.local
  4. Database: Set up MongoDB connection
  5. Authentication: Configure Clerk authentication
  6. Storage: Set up AWS S3 bucket
  7. Start: Run npm run dev to start development server
  8. Deploy: Follow deployment guides for AWS EC2 or Vercel
üí° Pro Tip: For production use, AWS EC2 is recommended for better performance and no file size limitations. Vercel has a 10-second function timeout that may affect some AI processing operations.
